# 🎓 Attenzo – Student Attendance Tracker

##  Overview

Attenzo is a full-stack web application developed to efficiently manage student attendance in an academic environment.

##  Features

* Student attendance tracking system
* User-friendly interface for students and admin
* Responsive design for multiple devices
* Clean and structured UI

##  Tech Stack

**Frontend:**

* HTML, CSS, Bootstrap, JavaScript

**Backend:**

* Django

**Database:**

* SQLite
